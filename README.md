# Rock, Paper, Scissors (Python)

A simple command-line Rock, Paper, Scissors game built with Python.

## Description
This program lets you play Rock, Paper, Scissors against the computer. It includes:

- Emoji support for better visuals
- Input validation for incorrect choices
- Score tracking for wins and losses
- The ability to play multiple rounds

## How to Run
1. Make sure Python 3 is installed
2. Download or clone the repository
3. Open a terminal or command prompt
4. Run the script using Python
